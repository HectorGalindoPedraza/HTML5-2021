Etiqueta bloquote
Se utiliza para marcar las citas a otros autores o documentos. Se puede incluir el atributo cite para poner un enlace al documento original o fuente 
            <blockquote cite="Link de donde se saco la informacion">
                 Lorem ipsum, dolor sit amet consectetur adipisicing elit. 
                  Eum obcaecati eligendi labore aspernatur nemo impedit 
                  nesciunt veritatis harum, dicta id!  
                 Labore modi libero incidunt consectetur vitae ex, expedita eveniet amet.
            </blockquote>
